Evolutionary Conservation of BRCA1 Across Primate Species Bioinformatics

Comparative bioinformatics analysis of BRCA1 protein sequences across primate species to assess evolutionary conservation of functional domains.

  • Curated BRCA1 protein sequences from NCBI and performed multiple sequence alignment using COBALT and Jalview
  • Quantified conservation across RING, linker, and BRCT domains using pairwise identity matrices
  • Constructed neighbour-joining phylogenetic trees with bootstrapping to evaluate evolutionary divergence
  • Interpreted conservation patterns in the context of functional constraint and disease-associated domains

Radio Calico In Progress

A browser-based internet radio player with lossless HLS audio streaming, real-time track metadata, album artwork, song history, and a user rating system — no subscriptions, no ads, no account required.

  • Live HLS streaming via hls.js served through AWS CloudFront CDN
  • Real-time now-playing metadata (artist, title, album, year) with progress tracking
  • Album artwork proxy with caching and MusicBrainz API integration
  • Persistent thumbs up/down rating system backed by SQLite
  • Full-stack: Express (Node.js) web server + Flask (Python) backend API
